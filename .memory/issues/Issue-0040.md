Decision:
中文 cookbook 增加“Issue 与 Evidence 示例”，把 issue card、execution log、review checkpoints、evidence entry 写成中文样例。

Reason:
在已有：
- 交付物写作指南
- review note 示例

之后，学习者仍然缺少治理类产物的具体样例。

如果不知道：
- issue card 怎么写
- evidence entry 怎么留

就很难把“学会一块内容”真正沉淀成可复查的工程证据。

Future:
把 issue / evidence 示例继续接回 month / week 实战清单
补 scorecard / review packet 示例
补 evidence entry 到 review note 的互链
