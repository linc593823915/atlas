Decision:
中文 cookbook 增加“Scorecard 与 Review Packet 示例”，把月度评分卡、月度 review packet、季度 replay 产物写成中文样例。

Reason:
在已有：
- review note 示例
- issue / evidence 示例

之后，学习者仍然容易卡在更高一层的验收材料：
- scorecard
- monthly review packet
- quarter replay

如果这些产物没有示例，学习者知道流程顺序，也很难自己写出像样的月度 / 季度结论。

Future:
把 scorecard 示例继续接回月度实战清单
补 quarterly replay 与年度答辩材料的互链
补更细的评分解释示例
