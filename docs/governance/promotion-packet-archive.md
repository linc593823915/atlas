# Promotion Packet Archive

Last updated: 2026-06-29

## Purpose

This archive defines what should be preserved for promotion-readiness review over time.

## Required Packet Contents

- annual review memo
- annual readiness matrix
- latest monthly scorecards
- latest quarterly KPI sheets
- strongest Atlas evidence artifacts
- promotion committee notes

## Archive Rules

- keep one packet per major readiness checkpoint
- preserve both strongest and weakest evidence
- link every claim to a concrete artifact
