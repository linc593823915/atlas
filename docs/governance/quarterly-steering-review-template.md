# Quarterly Steering Review Template

Last updated: 2026-06-29

## Purpose

Use this template to run a quarterly steering review across execution, Atlas progress, readiness, and risk.

## Inputs

- quarterly KPI sheet
- quarter replay pack
- latest three monthly scorecards
- current issue board snapshot
- current Atlas backlog state

## Review Agenda

1. quarter objective review
2. Atlas milestone review
3. metrics and evidence review
4. readiness and gap review
5. committee decision

## Review Output

- quarter status:
- major wins:
- major risks:
- intervention decision:
- next-quarter focus:

## Steering Questions

1. What improved most this quarter?
2. What remained flat despite visible effort?
3. What is the highest-value correction for the next quarter?
4. Is the current roadmap still the right roadmap?
