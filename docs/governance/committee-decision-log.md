# Committee Decision Log

Last updated: 2026-06-29

## Purpose

This log records decisions made by steering or promotion committees so follow-through is not lost.

## Decision Entry Template

- date:
- committee:
- decision:
- rationale:
- evidence cited:
- owner:
- follow-up action:
- next review date:

## Decision Types

- continue as planned
- accelerate scope
- reduce scope
- add mentor support
- delay promotion
- prepare for promotion review

## Rule

- no committee decision should exist without a named owner and next review date
