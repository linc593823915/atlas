# Exec Dashboard Pack

Last updated: 2026-06-29

## Purpose

This pack defines the minimum executive-facing dashboard view for the program.

## Dashboard Panels

### 1. Program Health

- current quarter
- monthly OKR pass rate
- quarter milestone completion
- blocked issue trend

### 2. Atlas Health

- subsystem progress
- reliability evidence count
- benchmark readiness
- documentation freshness

### 3. Talent Health

- readiness signal
- interview readiness trend
- strongest growth area
- biggest risk area

### 4. Governance Health

- review closure rate
- evidence completeness
- replay pack completion
- committee actions open

## Executive Questions

1. Is the program producing real engineering growth?
2. Is Atlas becoming a stronger platform?
3. Is the learner approaching the target role credibly?
4. Where is executive intervention needed?
