## Daily Assessment Standard

- Score out of 20.
- Read evidence: 5 points.
- Write artifact quality: 5 points.
- Build or lab progress: 5 points.
- Output clarity and evidence capture: 5 points.
- Passing bar: 12+ and the planned artifact exists.
- Strong bar: 16+ with explicit trade-offs and one improvement idea beyond the task.

## Interview Questions

1. How would you review this work using code-health criteria rather than preference?
2. What documentation or runbook changes are required before this is truly done?
3. What feedback would you give the author if this were a teammate’s change?
