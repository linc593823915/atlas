## 今日核心知识点

- 多 Tool routing 需要清楚的选择策略，而不是注册多个工具后随缘调用。
- 一个 tool 失败不应该把整条链拖崩，runtime 必须定义可退化路径。
- 今天要特别保护的不变量是：每一次 routing 决策和 tool failure 都必须事后可追溯。
- 如果 contract 写不清，本周最可能爆炸的点是：工具越来越多，但没有任何可审计、可解释的选择逻辑。
