## 今日核心知识点

- Agent role 要回答模型负责什么、runtime 负责什么、tool 负责什么，而不是只写一段 prompt。
- tool inventory 应该按外部能力组织，而不是把 repo 里的每个 helper 都变成 tool。
- 本周最大风险是：做出一个能聊的 Demo，却没有任何可复用的 contract。
- 今天至少要把“Agent RFC”的验证口径写出来。
