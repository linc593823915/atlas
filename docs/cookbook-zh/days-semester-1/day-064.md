## 今日核心知识点

- structured output 是 LLM 文本和确定性代码之间的桥梁，没有 schema 就没有稳定接口。
- schema 校验不只要判断对错，还要定义 reject、repair、retry 或 fail-closed 的后续动作。
- 本周最大风险是：把“差不多符合预期”的自然语言当成可靠数据直接使用。
- 今天至少要把“Agent 怎么调 Tool”的验证口径写出来。
