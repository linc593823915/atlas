## 今日核心知识点

- structured output 是 LLM 文本和确定性代码之间的桥梁，没有 schema 就没有稳定接口。
- 第一条 tool path 应该把输入、tool result 和最终输出结构串成一个完整闭环。
- 今天要特别保护的不变量是：任何进入后续逻辑的结构化输出都不能绕过 schema 验证。
- 如果 contract 写不清，本周最可能爆炸的点是：把“差不多符合预期”的自然语言当成可靠数据直接使用。
