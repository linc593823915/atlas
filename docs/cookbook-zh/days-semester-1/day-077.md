## 今日核心知识点

- 多 Tool routing 需要清楚的选择策略，而不是注册多个工具后随缘调用。
- routing 逻辑必须可解释，否则 operator 无法理解为什么模型做出某次工具选择。
- 本周最该沉淀的证据是：routing path、fallback case 和 tool negative tests。
- 如果这条风险不继续处理，后面最容易爆的是：工具越来越多，但没有任何可审计、可解释的选择逻辑。
