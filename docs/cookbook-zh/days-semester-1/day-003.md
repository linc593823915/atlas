## 今日核心知识点

- 学习系统本身也要有证据链：日志、Issue、架构笔记和复盘记录，后续每个月都要依赖这些资产。
- 最小模块边界至少要分清入口层、bootstrap 层、runtime 层和基础设施层，否则后续 Agent 能力会直接长在入口代码上。
- 实现阶段必须守住：任何新入口都应该复用同一套 bootstrap / shutdown 流程。
- 今天写的每一段代码都要服务于证据“至少留下 RFC 草稿、包结构图和学习记录入口。”。
