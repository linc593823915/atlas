## 今日核心知识点

- request dispatch 需要在保留 protocol context 的同时安全接入 Atlas runtime。
- 启动路径要把 capability registration、健康信号和优雅关闭都讲清楚。
- 今天最该盯住的风险仍然是：把 protocol bootstrap 当成 [`main()`](../atomic-concepts.md#concept-main-function) 的薄壳，结果 session lifecycle 和 runtime lifecycle 互相打架。
- 扩展之后仍然不能破坏：session 生命周期不能和依赖生命周期相互冲突。
