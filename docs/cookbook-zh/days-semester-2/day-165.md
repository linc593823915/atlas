## 今日核心知识点

- guardrail 是 runtime 规则：哪些动作禁止、哪些动作需确认、哪些输出必须收敛。
- 当策略信心不足时，系统应该可预期地降级到人工 review。
- 今天最该盯住的风险仍然是：把高风险动作偷偷塞进 agent autonomy，既没有审批点也没有人工接管点。
- 扩展之后仍然不能破坏：敏感动作在执行前必须可以被阻断，而不是事后补日志。
