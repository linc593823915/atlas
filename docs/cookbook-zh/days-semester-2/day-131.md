## 今日核心知识点

- 本周必须留下的证据是：bootstrap path、transport model 和 startup sequence。
- 今天验证最该覆盖的风险是：把 protocol bootstrap 当成 [`main()`](../atomic-concepts.md#concept-main-function) 的薄壳，结果 session lifecycle 和 runtime lifecycle 互相打架。
- 测试不能破坏的不变量是：session 生命周期不能和依赖生命周期相互冲突。
- 启动路径要把 capability registration、健康信号和优雅关闭都讲清楚。
