## 今日核心知识点

- runtime responsibility 要把协调逻辑从单个 agent 行为里分出来，否则 orchestration 无法测试。
- handoff boundary 要定义上下文、权限和证据在 agent 之间如何转移。
- 实现阶段必须守住：每一次 handoff 都必须保留目标 ownership、上下文来源和结果责任。
- 今天写的每一段代码都要服务于证据“multi-agent runtime RFC、role map 和 handoff boundary。”。
