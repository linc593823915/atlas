## 今日核心知识点

- handoff 触发条件必须显式定义，而不是靠“感觉另一个 agent 也许更合适”。
- approval path 要讲清 requester、approver、scope、timeout 和批准后的审计证据。
- 本周最大风险是：把高风险动作偷偷塞进 agent autonomy，既没有审批点也没有人工接管点。
- 今天至少要把“可解释的 handoff 流程”的验证口径写出来。
