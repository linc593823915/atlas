## 今日核心知识点

- 本周必须留下的证据是：handoff triggers、approval path 和 guardrail 规则。
- 今天验证最该覆盖的风险是：把高风险动作偷偷塞进 agent autonomy，既没有审批点也没有人工接管点。
- 测试不能破坏的不变量是：敏感动作在执行前必须可以被阻断，而不是事后补日志。
- 当策略信心不足时，系统应该可预期地降级到人工 review。
