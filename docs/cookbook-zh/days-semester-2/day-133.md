## 今日核心知识点

- MCP server bootstrap 要定义 session 如何启动、关闭以及如何拿到共享依赖。
- 启动路径要把 capability registration、健康信号和优雅关闭都讲清楚。
- 本周最该沉淀的证据是：bootstrap path、transport model 和 startup sequence。
- 如果这条风险不继续处理，后面最容易爆的是：把 protocol bootstrap 当成 [`main()`](../atomic-concepts.md#concept-main-function) 的薄壳，结果 session lifecycle 和 runtime lifecycle 互相打架。
