## 今日核心知识点

- 统一入口的意义是让工具行为可治理、可观测、可测试，而不是只少写几行接线代码。
- schema、timeout 和 audit 必须一起考虑，因为它们共同定义了一次 tool call 的运行契约。
- 今天最该盯住的风险仍然是：把 gateway 写成一个薄路由器，实际上没有收回任何控制权。
- 扩展之后仍然不能破坏：每一次 tool invocation 都必须走过同一条可检查、可拦截的 gateway path。
