## 今日核心知识点

- handoff 触发条件必须显式定义，而不是靠“感觉另一个 agent 也许更合适”。
- guardrail 是 runtime 规则：哪些动作禁止、哪些动作需确认、哪些输出必须收敛。
- 今天要特别保护的不变量是：敏感动作在执行前必须可以被阻断，而不是事后补日志。
- 如果 contract 写不清，本周最可能爆炸的点是：把高风险动作偷偷塞进 agent autonomy，既没有审批点也没有人工接管点。
