## 今日核心知识点

- stateful tests 要模拟重启、重复 resume、过期 checkpoint 和分支失败。
- 长流程 drill 需要覆盖 operator takeover、卡死状态和 checkpoint 损坏后的恢复决策。
- 本周最大风险是：只验证本地 happy-path resume，从未演练真实恢复流程。
- 今天至少要把“工作流测试集”的验证口径写出来。

## 每日评分标准

- 满分 `20` 分。
- 阅读证据：5 分。
- 写作产物质量：5 分。
- 构建或实验推进：5 分。
- 输出清晰度与证据留存：5 分。
- 及格线：`12+`，且今天计划产物真实存在。
- 优秀线：`16+`，并能清楚说明取舍与下一步改进。
