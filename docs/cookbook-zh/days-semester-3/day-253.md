## 今日核心知识点

- prompt injection 防御首先要识别不可信指令从哪里进入系统。
- 数据泄漏路径不仅在模型输出，也在 tool result、trace、log、memory 和人工 review 面。
- 本周最大风险是：以为 prompt hygiene 已经等于治理，忽略了 tool、trace 和数据面的攻击面。
- 今天至少要把“一组 adversarial case”的验证口径写出来。
