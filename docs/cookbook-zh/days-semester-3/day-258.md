## 今日核心知识点

- prompt injection 防御首先要识别不可信指令从哪里进入系统。
- 好的防御证据既要证明恶意流量被拦住，也要证明正常流量没有被误伤。
- 文档和 review 要服务于证据：injection tests、leakage checks 和 misuse results。
- 接手者最容易踩到的坑通常就是：以为 prompt hygiene 已经等于治理，忽略了 tool、trace 和数据面的攻击面。

## 每日评分标准

- 满分 `20` 分。

- 及格线：`12+`，且今天计划产物真实存在。
