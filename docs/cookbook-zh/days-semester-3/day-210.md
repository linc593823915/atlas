## 今日核心知识点

- stateful tests 要模拟重启、重复 resume、过期 checkpoint 和分支失败。
- 月度复盘要说明 durable model 是真正可运行的，还是只在理论上可恢复。
- 本周最该沉淀的证据是：stateful tests、recovery drills 和 workflow docs。
- 如果这条风险不继续处理，后面最容易爆的是：只验证本地 happy-path resume，从未演练真实恢复流程。
